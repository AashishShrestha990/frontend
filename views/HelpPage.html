Smart Project Visualization

‘Smart Project Visualization’ is web-application platform where an individual user can create a project and manage the project with the help of different tools, techniques and techniques available. The application does not have any restriction on what they can use. The application have modest interface that can be effortlessly used by the user. You can create a project Name it, Provide description for the project and adjust the progress bar to track progression.

Along with a project you can make note along with important images files whick you can keep track of for further use. You can easily manage, monitor and modify canages in the memo

How to use?

The app provides with a side Navigation bar whic includes links to the pages you want to go. Simply click on the text or near it to go that page. If you want to create a project click on "Start a project" and a form will appear to add a project You can view the Project statistics in the dashboard or in "project Stat" tab. collective summary chart of your projects is visualized in caharts.
